# AI Slack Bot
A starter submission for the Stardance YSWS event by HackClub, this bot is very simple, it uses the HackClub AI API to reply to questions given as an argument to /asb-ask

Available commands:
/asb-ping - Check if the bot is working and the latency
/asb-joke - Fetch a random joke from the Official Joke API
/asb-calc [expression] - Calculate a mathematical expression
/asb-ask [question] - Ask the AI a question
/asb-whoami - Info about the bot and its developer
/asb-quote - Random quote

oh and asb stands for AI Slack Bot

### AI USAGE DECLARATION:
AI's help was used in the development of this project to help me parse the JSON response coming from the AI API as I'm not experienced in JSON parsing via JavaScript.
